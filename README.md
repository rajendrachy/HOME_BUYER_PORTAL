# 🏠 Home Buyer Portal - Nepal Government Initiative

A comprehensive MERN stack web application that helps first-time home buyers in Nepal apply for government subsidies and get bank loans.

## 🌐 Live Demo

- **Frontend:** https://home-buyer-portal.vercel.app
- **Backend API:** https://home-buyer-portal-1.onrender.com

## 📋 Project Overview

The Home Buyer Portal digitizes the entire process of applying for government home buyer subsidies and bank loans in Nepal. It eliminates paperwork, reduces processing time, and provides transparency to citizens.

### Problem Solved
- First-time home buyers don't know about government subsidies
- Manual paperwork and middlemen cause delays
- Long waiting times and no transparency
- Difficulty comparing bank loan offers

### Solution
- 100% digital, paperless application process
- Real-time application tracking
- Compare offers from multiple banks
- 15-day processing guarantee

## 🚀 Features

### For Citizens
- ✅ Check eligibility for government subsidy
- ✅ Submit online applications
- ✅ Track application status in real-time
- ✅ View and accept bank loan offers
- ✅ View application history

### For Municipality Officers
- ✅ View all applications in their municipality
- ✅ Review and approve/reject applications
- ✅ Set subsidy amounts (up to 15%)
- ✅ Add officer notes

### For Bank Officers
- ✅ View approved applications
- ✅ Submit loan offers with EMI calculation
- ✅ Manage offers
- ✅ View offer history

### Public Features
- ✅ Track any application by ID (no login required)

## 🛠️ Technology Stack

### Frontend
- **React 18** - UI Framework
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **Axios** - API calls
- **React Router DOM** - Navigation

### Backend
- **Node.js** - Runtime
- **Express.js** - Web framework
- **MongoDB Atlas** - Cloud database
- **JWT** - Authentication
- **bcryptjs** - Password encryption
- **Multer** - File upload (optional)
